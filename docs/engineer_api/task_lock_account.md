title: 执行任务：锁定账户
sidebar_position: 13
---

接口描述：执行任务：锁定当前登录的电脑账户，类似于移动设备的熄屏（此处仅提供简单说明，具体效果等请看流程树节点设置）。

## 请求说明
|HTTP Method|HTTP URL|
|:-:|:-:|
|POST|http://localhost:7902/Api/task/oe|


## 请求参数

:::info

不需要携带任何请求参数（作用效果：锁定当前登录的电脑账户，类似于移动设备的熄屏）。

:::

## 返回参数

|属性|类型|说明|
|:-:|:-:|:-:|
|code|Number|错误码（code == 0 表示成功）|
|msg|String|错误发生原因|
