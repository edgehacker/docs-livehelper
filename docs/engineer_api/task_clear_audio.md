---
title: 执行任务：移除音频
sidebar_position: 10
---

接口描述：执行任务：停止并移除所有音频，包括语音合成（但不包括绿幕视频）（此处仅提供简单说明，具体效果等请看流程树节点设置）。


## 请求说明
|HTTP Method|HTTP URL|
|:-:|:-:|
|POST|http://localhost:7902/Api/task/o9|


## 请求参数

:::info

不需要携带任何请求参数（作用效果：停止并移除所有音频，包括语音合成（但不包括绿幕视频））。

:::

## 返回参数

|属性|类型|说明|
|:-:|:-:|:-:|
|code|Number|错误码（code == 0 表示成功）|
|msg|String|错误发生原因|
